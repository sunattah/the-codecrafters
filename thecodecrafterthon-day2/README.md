# HOW IT RUNS #

the code runs like this:

```go
go run main.go
```
after words you will see where you will enter the value if you input any value be it hex or bin or dec it will convert every thing to the value it is to be converted to
e.g

hex >> ff - 255
bin >> 1010 - 10
dec >> 255
bin : 11111111
hex FF
