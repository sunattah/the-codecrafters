# GO

* Go is a statically typed compiled programming language created in 2007 by microsoft and released publicly in 2009. It is known for it's simplicity. It is opensource and crossplatform and it is used to create high perfprmance applications.
* Go can be used in various areas such as Bankend development, cross-platform applications, cloud-native developments.
* Go is easy to learn, runs fast at compile time, supports concurency and memory management and different platform compatibility.
