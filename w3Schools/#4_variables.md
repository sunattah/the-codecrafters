# Variables

* Variables serve as containers for holding values. A variable consist basically of 3 parts, the var keyword, the variable name, the variable type.

## Declaring a varaible 
* Declaring a variable can be done in two ways. using the "var" keyword or with the ":=" sign.

```go 

package main


var name string = "Azi" // Declearing a variable using the "var" keyword.

func main() {

    num1 := 1 // Declearing a variable using the ":=" sign.
}

```
